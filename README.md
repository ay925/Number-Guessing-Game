# Number Guessing Game 🎯

A simple Android game built using Kotlin and Jetpack Compose.

## Features
- Random number generation
- Input validation
- Score tracking

## Tech Stack
- Kotlin
- Jetpack Compose

## How to Run
1. Clone repository
2. Open in Android Studio
3. Run on emulator or device
